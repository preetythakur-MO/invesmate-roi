## 🚀 Deploy to Vercel in 2 Minutes (No Code Needed)

### Option A — Drag & Drop (Easiest)

1. Go to **https://vercel.com** and sign up / log in (free account)
2. On the dashboard, click **"Add New → Project"**
3. Click **"Browse"** or drag the entire `invesmate-roi` folder into the upload area
4. Click **Deploy**
5. In ~30 seconds you get a live link like: `https://invesmate-roi.vercel.app`
6. Share this link with your team and upper management ✅

### Option B — Via Vercel CLI

```bash
npm install -g vercel
cd invesmate-roi
vercel --prod
```
